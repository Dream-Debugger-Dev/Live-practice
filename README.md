Welcome to **Neon-Vault**, a curated collection of retro-inspired web experiences. This project is a living laboratory where I experiment with CSS animations, interactive UI, and the nostalgia of the 80s arcade era.

## 🚀 Featured Project: Retro Arcade Haven
The current flagship of this repo is a **Pac-Man Tribute Page**, featuring:
* **Neon Aesthetics:** High-contrast green and orange color palette.
* **Interactive Cards:** "View Details" functionality with smooth transitions.
* **Responsive Slider:** A game collection carousel showcasing various titles.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for accessibility.
* **CSS3:** Custom animations and layout (Flexbox/Grid).
* **JavaScript:** Handling interactive elements and navigation.
* **CodeSandbox:** Primary development environment.
